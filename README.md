Example usage:

```
https://timestampmicro-ani007.c9users.io/December%2015,%202015
```
```
https://timestampmicro-ani007.c9users.io/1450137600
```

Example output:
```
{ "unix": 1450137600, "natural": "December 15, 2015" }
```
